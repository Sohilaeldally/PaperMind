# PaperMind

A specialized RAG (Retrieval-Augmented Generation) system for understanding and answering questions about AI/ML research papers.

## Current Status
🚧 Work in progress.

## Pipeline
- ✅ Upload & Parsing (PDF/DOCX/TXT)
- ✅ Chunking
- ✅ Embedding (multilingual-e5-small + pgvector)
- 🔲 Search Endpoint
- 🔲 RAG (Question → Answer)
- 🔲 Structure-aware Chunking for academic papers
